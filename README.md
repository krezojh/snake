# Snake Portal Game

## 프로젝트 소개

Python과 pygame으로 만든 Snake Game입니다.  
기본 Snake Game에 포탈 기능을 추가해 이동 전략 요소를 넣었습니다.

플레이어는 뱀을 조작해 먹이를 먹고 점수를 얻습니다.  
뱀이 벽이나 자기 몸에 부딪히면 게임이 종료됩니다.

## 게임 목표

뱀을 최대한 오래 생존시키며 높은 점수를 얻는 것이 목표입니다.

## 조작 방법

- 방향키 또는 WASD: 이동
- Space: 일시정지 / 재개
- R: 게임 재시작
- ESC: 게임 종료

## 주요 기능

- 시작 화면
- 게임 진행 화면
- 일시정지
- 게임 오버
- 점수 표시
- 최고 점수 표시
- 먹이 획득 시 점수 증가
- 먹이 획득 시 뱀 길이 증가
- 벽 충돌 판정
- 자기 몸 충돌 판정
- 포탈 이동 기능

## 포탈 기능

화면에는 서로 연결된 두 개의 포탈이 생성됩니다.

뱀이 한쪽 포탈에 들어가면 반대쪽 포탈에서 같은 방향으로 나옵니다.  
포탈을 통과하면 추가 점수를 얻을 수 있습니다.

포탈은 다음 위치와 겹치지 않도록 생성됩니다.

- 뱀의 몸
- 먹이
- 벽

먹이를 일정 개수 먹을 때마다 포탈 위치가 새로 바뀝니다.

## 사용 기술

- Python
- pygame
